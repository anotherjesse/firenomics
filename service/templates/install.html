Installation

To get started you have to install another extension! By installing our extension we will (eventually) automatically update your extension list whenever you install, uninstall or disable an extension.

Install:

Install by clicking here, then restart your browser as requested.

Manual Update (for now)

Currently updating is manual. To get started click Update Firenomics in the Tool Menu of your browser. You will be asked to login with a Google Account. If this is your first time using Firenomics you will be asked to choose a username.

Pre-Alpha !!!

Firenomics is very new. Please try it out and give feedback on uservoice. If you are new to addons, this is probably not the first extension for you :)
